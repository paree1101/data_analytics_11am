# What is python?
1. Python is a cross-platform language 
2. It is the most popular cross-platform langauge 
3. It's an interpreter based langauge
4. When executed, it is checked step by step by an interpreter
5. Interpreter also converts high level langaue into low level langauge (in binary 01)
6. Python is used to create web apps, machine learning, data analytics, AI, data science, agentic AI, IoT
7. Easy to learn 
8. Python provides some packages or libraries that is in the form of bundle of code

    **how to use libraries**
    ```
    import os from os
    ```
9. Python is a light-weight programming langauge
10. It provides some external packages and they can be used via download or install 
    **how to install a library or package**
    ***pip (python install)***
    ```
    pip install matplotlib
    pip install pandas
    pip install numpy
    pip install seaborn 
    pip install excel
    ```

11. python files are saved using extension .py
12. how to check version of python
    ```
    python --version
    ```
13. how to check version of pip
    ```
    pip --version
    ```
14. how to upgrade pip
    ```
    pip install --upgrade pip
    ```
